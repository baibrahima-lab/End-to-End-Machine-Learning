## 🧬 Pipeline & Innovation

### 1. Rectification de Signal 🧼
* **Interpolation Dynamique :** Correction des 1,25% de données manquantes sans briser la saisonnalité.
* **Fusion Temporelle :** Création d'un index continu par minute pour une précision maximale.

### 2. Modélisation Multi-Horizon 📈
* ⏱️ **Modèle Minute :** Réactivité en temps réel pour Smart Meters.
* 📅 **Modèle Day-Ahead :** Gain de **+18.9% de précision** pour l'optimisation des marchés de gros.
